# commit

Read **`.cursor/prompts/git-conventions-workflow.md`** (sections **Commit messages** and **Creating a commit**).

Inspect **`git status`** and **`git diff`** (staged first; include unstaged if the user is committing the working tree). Propose a commit **title** and optional **minimal body** per those conventions.

Do **not** run **`git commit`** unless the user explicitly asks you to execute it.
