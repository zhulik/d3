# pr

Read **`.cursor/prompts/git-conventions-workflow.md`** (sections **Pull requests** and **Opening a PR**) and **`.github/pull_request_template.md`**.

Draft a **full PR description** in Markdown, matching the template headings and checklists. Use the current branch name and **`git log` / `git diff`** against the default base branch (e.g. `main`) when helpful.

Do **not** run **`gh pr create`** or open a browser unless the user explicitly asks.
